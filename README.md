# Projet7_Frontend

## Project setup

Accédez sur https://nodejs.org/en/ pour télécharger puis installer Node.js sur votre machine.

Ensuite installer NPM avec cette ligne de commande :
```
npm install
```

### Start server
```
npm run serve
```
